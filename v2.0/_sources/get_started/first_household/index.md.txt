# Your first household

Hands-on tutorial for registering your first household in OpenSPP.

**For:** New users and registry staff

This tutorial walks you through creating a household in the OpenSPP registry, from accessing the registry to adding family members.

## What you'll learn

By the end of this tutorial, you will have:

- Navigated the Registry module
- Created a new household (group)
- Registered individual family members
- Linked individuals to the household with their roles

## Tutorial steps

This tutorial covers the following steps:

1. **Access the registry** - Navigate to the Registry module and understand the interface
2. **Create a household** - Register a new household group
3. **Add members** - Create individuals and link them to the household

## Prerequisites

Before starting this tutorial:

- OpenSPP should be installed and running (see {doc}`../installation/index`)
- OpenSPP SPMIS should be installed
- Sample data loaded for exploration (see {doc}`../explore/index`)
- Access to OpenSPP with Registry Officer or Administrator privileges
