# Reference

**Applies to:** Social Registry, SP-MIS, DRIMS

## What You'll Find Here

This section provides quick reference material for **users** who need to look up specific information while working in OpenSPP.

Use these reference guides when you need to:

- Understand what geographic areas are and how they work
- Look up vocabulary codes (gender, marital status, relationship types, etc.)
- Find the meaning of a specific code or value in a dropdown

## Reference vs User Guides

**Reference guides** help you look up specific information quickly. Use them when:

- You need to know what a specific code means
- You want to understand how areas are organized
- You're looking for valid values for a field

**User guides** help you complete specific tasks. Use them when:

- You need to register a new beneficiary
- You want to enroll someone in a program
- You're processing a payment
