Getting started#

Welcome to OpenSPP! This guide will help you get your social protection platform up and running quickly.

Quick start path#

Start here if you're setting up OpenSPP for the first time:

I want to use the OpenSPP products

  1. Install OpenSPP on Debian/Ubuntu (30 minutes)


  1. Install SP-MIS, Social Registry, or Farmer Registry from the available modules.


  1. Get your system ready for use:

    • Create your first program - Creating a Program

    • Import beneficiary data - Registry Management

    • Configure user access - Administration

I want to select my modules freely

  1. Install OpenSPP on Debian/Ubuntu (30 minutes)


  1. Choose modules to install. Read more about module installation and available modules


  1. Get your system ready for use:

    • Create your first program - Creating a Program

    • Import beneficiary data - Registry Management

    • Configure user access - Administration

OpenSPP configurations#

OpenSPP offers three primary configurations based on your needs:

Configuration

Best For

Key Features

SP-MIS

Social protection programs, cash transfers, humanitarian aid

• Registry management
• Program cycles
• Eligibility targeting
• Payment processing

Social Registry

Centralized beneficiary database, inter-program coordination

• Unified beneficiary data
• Deduplication
• Data sharing across programs
• Needs assessment

Farmer Registry

Agricultural support, farmer subsidies, rural development

• Farm mapping
• Crop tracking
• Input distribution
• Seasonal cycles
