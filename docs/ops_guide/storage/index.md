# Storage

**For:** System administrators configuring storage systems

This guide covers file and document storage configuration in OpenSPP, a new section introduced in V2 to support enhanced document management capabilities.

**NEW V2 SECTION**: Storage configuration is a new operational area in OpenSPP V2.

## Topics Covered

- **File Storage**: Local filesystem storage configuration
  - Storage paths and permissions
  - Disk space management
  - File organization strategies
- **Object Storage**: Cloud and S3-compatible storage (NEW V2)
  - Amazon S3 configuration
  - Azure Blob Storage setup
  - MinIO and other S3-compatible providers
  - Storage buckets and access policies
- **Document Management**: Configuring the document management system
  - Document types and categories
  - Attachment handling
  - Version control
  - Retention policies
- **Storage Optimization**: Performance tuning and cost optimization
  - Caching strategies
  - Storage tiers (hot/cold)
  - Compression options
