# Modules Index

This page provides a comprehensive listing of all OpenSPP modules, organized by functional category. OpenSPP currently has approximately 82 modules that extend the platform's capabilities.

## Module Categories

Below is a preview of the module categories. Each category groups related modules by their primary function.

### Core Modules
Foundation modules that provide essential platform functionality.

### Registry Modules
Modules for managing registrants, individuals, groups, and registry operations.

### Programs Modules
Modules for program management, eligibility, enrollment, and cycles.

### Payments Modules
Modules for payment processing, disbursements, and financial management.

### Integration Modules
Modules for external system integration and data exchange.

### GIS Modules
Geographic Information System modules for spatial data and mapping.

### Security Modules
Modules for access control, encryption, and security features (V2 enhanced).

### Utilities Modules
Helper modules providing common utilities and tools.
