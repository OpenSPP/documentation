***************************************
Program Manager
***************************************
The program manager determines which registrants/groups are included in the program.

The ``g2p_programs.models.managers.program_manager`` module provides the classes that define
the interface for this manager. ``BaseProgramManager`` defines the abstract base class,
and ``DefaultProgramManager`` is the default implementation.
