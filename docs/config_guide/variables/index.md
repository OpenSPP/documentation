# Variables & Indicators

**For: Implementers**

**NEW V2**

The unified variable system replaces the previous separate indicators module. Variables store calculated values, cached data, and program indicators with support for different period granularities and automatic cache management.

## Topics Covered

- **Overview** - Introduction to the unified variable system
- **Variable Types** - Different types of variables and their uses
  - Cached calculations
  - Program indicators
  - Derived values
  - External data references
- **Caching** - Automatic cache management and refresh strategies
- **Period Granularity** - Daily, monthly, quarterly, and annual periods
- **Variable Configuration** - Defining variables in OpenSPP Studio
- **Using Variables** - Referencing variables in CEL expressions
