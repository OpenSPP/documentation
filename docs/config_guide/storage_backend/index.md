# Storage backend

This guide is for **implementers** configuring where OpenSPP stores uploaded files and attachments.

## What you'll find here

- **{doc}`overview`** - Backend types, connection settings, and file storage policies

```{toctree}
:hidden:
:maxdepth: 1

overview
```

## Quick start

1. Navigate to **Settings > Storage Backends**
2. Review or create a **Storage Backend** (e.g., local filesystem, S3-compatible)
3. Configure the connection settings
4. Test the connection
5. Set as the active backend for file storage
