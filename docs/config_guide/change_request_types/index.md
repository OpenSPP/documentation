# Change Request Types

**For: Implementers**

**NEW V2**

Change request types define the workflows for modifying registrant data. Configure change request types, approval sequences, and validation stages to match your organization's governance requirements.

## Topics Covered

- **Overview** - Introduction to change request workflows
- **Configure Types** - Defining change request types in OpenSPP Studio
  - Field-level changes
  - Group membership changes
  - Program enrollment changes
  - Custom change types
- **Approval Sequences** - Multi-stage approval workflows
- **Validation Stages** - Automated validation and quality checks
- **Permissions** - Role-based access control for change requests
- **Audit Trail** - Tracking change history and accountability
