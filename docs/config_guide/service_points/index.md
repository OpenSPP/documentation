# Service points

This guide is for **implementers** configuring physical or virtual locations where beneficiaries receive services or collect benefits.

## What you'll find here

- **{doc}`overview`** - Service point concepts, creation, area assignment, and service types
- **{doc}`lifecycle`** - Enable, disable, and manage service point operational status

## Quick start

1. Navigate to **Service Points** in the main menu
2. Click **Create** to add a new service point
3. Enter the **Agent** name and assign an **Area**
4. Select **Service Types** from the vocabulary (e.g., "Cash-out point", "Distribution center")
5. Optionally link a **Company** for the managing organization
