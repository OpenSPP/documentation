
# Program management

This section offers comprehensive guides to help you create, manage, and deliver social protection programs effectively. From initial setup and beneficiary enrollment to fund allocation and cycle management, these documents provide step-by-step instructions to ensure your programs are well-defined and efficiently executed.

- {doc}`create_program`: Learn how to set up a social protection program from start to finish, including the configuration of eligibility criteria, cycle management, and entitlement managers.

- {doc}`enrol_beneficiaries`: This guide explains the process of enrolling beneficiaries into a program, either automatically during program creation or manually as a separate step.

- {doc}`create_cycle`: A comprehensive tutorial on creating and managing program cycles, as well as preparing and approving entitlements for distribution.

- {doc}`configure_entitlements`: This document provides a step-by-step guide to configuring cash entitlements for a social protection program.

- {doc}`allocate_funds`: This guide covers the essential steps for allocating funds to a program to ensure that entitlements can be successfully approved and disbursed.

- {doc}`using_geotargeting`: Learn how to use geotargeting to define and manage geographic areas, enabling more accurate and effective enrollment for your social protection programs.
