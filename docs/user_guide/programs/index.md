# Programs

**For:** Users (program staff, operators)

Programs are at the heart of OpenSPP, allowing you to create and manage social protection programs with flexible eligibility criteria, enrollment processes, and benefit distribution rules.

## Topics Covered

- **Create Programs** - Set up new social protection programs
- **Manage Cycles** - Configure and run program cycles
- **Enroll Beneficiaries** - Process beneficiary enrollment
- **Verify Eligibility** - Define and apply eligibility criteria
- **Program Configuration** - Configure program settings and parameters
- **Beneficiary Management** - Track and manage enrolled beneficiaries
- **Program Lifecycle** - Manage programs from creation to completion
