# Dashboards & Reports

**For:** Users (program staff, operators)

Dashboards and Reports provide analytics and visualization tools for monitoring program performance, registry statistics, and operational metrics. V2 includes enhanced chart types and new analytical capabilities.

## Topics Covered

- **Registry Overview** - View registry statistics and demographics
- **Program Overview** - Monitor program performance and metrics
- **Multi-program Comparison** - Compare metrics across multiple programs
- **Fairness Scorecard** - Analyze program fairness and equity metrics
- **Custom Dashboards** - Create personalized dashboards
- **Report Generation** - Generate and export reports
- **Data Visualization** - Use enhanced chart types (V2) for data analysis
- **Key Performance Indicators** - Track KPIs and program targets
