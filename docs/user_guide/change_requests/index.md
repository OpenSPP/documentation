# Change Requests

**For:** Users (program staff, operators)

Change Requests is a V2 feature that allows registrants to submit requests for changes to their data, and enables staff to review and process these requests in a structured workflow. This system provides transparency and accountability for data updates.

## Topics Covered

- **Submit Requests** - Create change requests on behalf of registrants
- **Review and Approve** - Process incoming change requests
- **Request Types** - Different types of changes (personal info, household composition, etc.)
- **Validation Rules** - Configure validation for different request types
- **Approval Workflows** - Multi-level approval processes
- **Request History** - Track all change requests and their outcomes
- **Notifications** - Configure notifications for request status updates
