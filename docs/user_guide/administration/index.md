# Administration

**For:** Users (program staff, operators)

Administration tools allow program managers to configure system settings, manage users and access permissions, and monitor system activity. V2 introduces a 3-tier access management system for enhanced security.

## Topics Covered

- **Users and Roles** - Create and manage user accounts and roles
- **Access Management (3-tier V2)** - Configure the three-tier access control system (program, area, individual)
- **Audit Logs** - Review system activity and changes
- **Consent Management** - Manage data consent and privacy settings
- **System Configuration** - Configure system-wide settings
- **Data Security** - Manage data access and security policies
- **User Permissions** - Define granular permissions for different roles
- **Activity Monitoring** - Track user actions and system events
