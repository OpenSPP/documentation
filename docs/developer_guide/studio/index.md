# Studio

**For: developers**

Learn how OpenSPP Studio provides no-code configuration and how to make your custom modules Studio-aware.

## Topics covered

- **Studio overview**: Field builders, logic packs, and configuration wizards
- **Studio-aware modules**: Making custom modules compatible with Studio configuration
- **Studio API V2**: Programmatic access to Studio configuration
