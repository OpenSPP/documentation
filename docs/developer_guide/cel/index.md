# CEL (Common Expression Language)

**For: developers**

Learn how OpenSPP uses the Common Expression Language (CEL) as its core expression system for scoring, eligibility, approval logic, and dynamic domain queries.

## Topics covered

- **CEL overview**: How CEL replaces the legacy indicator system
- **Variables and providers**: The unified variable system and value providers
- **Domain queries**: Building dynamic filter expressions with CEL
- **CEL widgets**: Frontend components for CEL expression editing
