# Testing

**For: developers**

Learn how to write and run tests for OpenSPP modules, including unit tests, integration tests, and end-to-end tests.

## Topics covered

- **Test framework**: Odoo test classes and OpenSPP test utilities
- **Unit tests**: Testing models, methods, and business logic
- **Integration tests**: Testing module interactions and workflows
- **End-to-end tests**: Playwright-based UI testing
- **Test data**: Managing test fixtures and demo data
