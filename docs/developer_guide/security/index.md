# Security and encryption

**For: developers**

Learn how OpenSPP handles PII encryption, key management, and security scanning. This section covers the cryptographic services available to module developers and security best practices.

## Topics covered

- **PII encryption**: Field-level encryption for personally identifiable information
- **Key management**: JWK/JWKS, key rotation, and pluggable key providers
- **Security scanning**: Semgrep, Gitleaks, Trivy, and CI/CD security tooling
