# Custom change request types

**For: developers**

Learn how to build complex change request types that go beyond what Studio supports. While Studio can create basic CR types, complex change requests with custom validation, multi-step approval logic, or specialized detail models require custom module development.

## Topics covered

- **When to use custom CR types**: Understanding the boundary between Studio-based and module-based CR types
- **Detail models**: Creating custom detail models for complex data capture
- **Validation logic**: Implementing custom validation rules
- **Approval workflows**: Building multi-step approval logic with custom conditions
