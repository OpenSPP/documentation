# Contributing

**For: developers**

Learn how to contribute to the OpenSPP codebase, including code style standards, pre-commit hooks, and the pull request workflow.

## Topics covered

- **Code style**: Ruff, Pylint, isort, and formatting standards
- **Pre-commit hooks**: Automated checks before committing
- **Pull request workflow**: Branch conventions, review process, and CI checks
